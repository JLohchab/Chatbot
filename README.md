# Rasa Auto Dealership Chatbot

This repository contains a Rasa-based chatbot for an auto dealership. The bot can assist customers with inquiries about vehicle availability, booking test drives, and providing information about financing options.

## Project Structure
- `actions/`: Contains custom actions for the chatbot.
- `data/`: Contains training data, NLU data, and stories.
- `config/`: Contains configuration files for the Rasa project.
- `domain.yml`: Defines the domain of the chatbot, including intents, entities, slots, and responses.
- `requirements.txt`: Lists the required packages for this project.

## Getting Started
1. Clone the repository.
2. Install the required packages using `pip install -r requirements.txt`.
3. Run `rasa train` to train the model.
4. Start the Rasa action server.
5. Start the Rasa server.
